# Chrome-Epub-Reader

A simple chrome extension to read epub files. Works offline too!

# Features
- Navigate using TOC menu
- Continuous reading mode that automatically loads sections as needed
- Dark mode
- Change the font size
- Vim like movements: j, k, gg, G
